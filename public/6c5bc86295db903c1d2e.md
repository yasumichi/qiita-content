## ユークリッドの互除法 Python 版

- [拡張ユークリッドの互除法 〜 一次不定方程式 ax + by = c の解き方 〜 - Qiita](https://qiita.com/drken/items/b97ff231e43bce50199a)

```python
def gcd(a, n):
    if n == 0:
        return  a
    else:
        return gcd(n, a % n)
```
