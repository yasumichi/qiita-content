## 概要

[GitBucket](https://gitbucket.github.io/)用のプラグイン [GitBucket Markdown Enhanced Plugin](https://github.com/yasumichi/gitbucket-markdown-enhanced)を開発しています。

GitBucket Markdown Enhanced Plugin は、GitBucket 標準のマークダウンレンダリングエンジンを置き換えるプラグインです。

目標は、[Visual Studio Code](https://code.visualstudio.com/) の [Markdown Preview Enhanced](https://shd101wyy.github.io/markdown-preview-enhanced/#/) 向けの markdown ファイルを軽易に Web で共有できる環境です。

今回は、プレゼンテーション機能の強化について、紹介します。

まだ、加筆するかもしれません。

## Reveal.js 用プラグインの追加

- [Reveal.js-plugin: Overview](https://rajgoel.github.io/reveal.js-demos/)

上記デモで使用されている以下のプラグインを追加しました。

- [Custom controls](https://github.com/rajgoel/reveal.js-plugins/tree/master/customcontrols)
- [Chalkboard](https://github.com/rajgoel/reveal.js-plugins/tree/master/chalkboard)

これにより、スライド上でペンが使用できるようになりました。

合わせて、[PDF Export](https://revealjs.com/pdf-export/)へ切り替えられるボタンを配置しました。

![](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/ca6869ce-73c8-45cc-8e4e-c1b209e9e65e.png)

上の画像のようにスライド左下にボタンが3つ配置されます。左から順に以下のような機能が呼ばれます。

- スライドを黒板に切り替える
- スライド上でペンによる描画を行う
- PDF Export モードに切り替える

![](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/cbd8d039-7cf3-4013-af54-1b946894ba99.png)

上の画像は、Chalkboard でスライドに描画した様子です。

![](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/8fc8aae1-af1c-418e-86b4-041fabac6105.png)

PDF Export モードに切り替えた場合は、上の画像のようにスライドショーに戻るためのアイコンのみ左下に表示されます。

## リポジトリ毎に Reveal.js のテーマを設定できるようにしました

以下の手順で Reveal.js のテーマを変更できます。

- リポジトリの `Settings` メニューを開く
- `Reveal` タグを開く
- `Theme` のドロップダウンリストから、使用したいテーマを選択
- `Apply changes` ボタンをクリック

![](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/142203ea-c8c7-49aa-b1c4-2a54765d0d71.png)
