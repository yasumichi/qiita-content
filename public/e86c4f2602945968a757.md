## GitBucket 4.45.0 リリース

- [GitBucket 4.45.0をリリースしました - たけぞう瀕死ブログ](https://takezoe.hatenablog.com/entry/2026/01/10/132525)
- [GitBucket 4.45.0 released!](https://gitbucket.github.io/gitbucket-news/gitbucket/2026/01/10/gitbucket-4.45.0.html)

[GitBucket 4.45.0](https://github.com/gitbucket/gitbucket/releases/tag/4.45.0) が 2026/01/10 にリリースされ、[GitBucket Markdown Enhanced Plugin](https://github.com/yasumichi/gitbucket-markdown-enhanced) が、以下の機能にも適用されるようになりました。

- Commit comments
- Issues
- Pull requests
- Wiki

GitBucket 4.45.0 に関する issue は、以下で確認できます。

[Issues](https://github.com/gitbucket/gitbucket/issues?q=is%3Aclosed+milestone%3A4.45.0)
