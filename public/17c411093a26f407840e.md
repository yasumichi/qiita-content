## 概要

[GitBucket](https://gitbucket.github.io/)用のプラグイン [GitBucket Markdown Enhanced Plugin](https://github.com/yasumichi/gitbucket-markdown-enhanced)を開発しています。

GitBucket Markdown Enhanced Plugin は、GitBucket 標準のマークダウンレンダリングエンジンを置き換えるプラグインです。

目標は、[Visual Studio Code](https://code.visualstudio.com/) の [Markdown Preview Enhanced](https://shd101wyy.github.io/markdown-preview-enhanced/#/) 向けの markdown ファイルを軽易に Web で共有できる環境です。

これまで [gitbucket-emoji-plugin](https://github.com/gitbucket/gitbucket-emoji-plugin/) で提供されている絵文字の内、[flexmark-java](https://github.com/vsch/flexmark-java) の [EmojiExtension](https://github.com/vsch/flexmark-java/wiki/Extensions#emoji) で対応していない絵文字については、変換することができませんでした。(例: `:109:`)

今回は、gitbucket-emoji-plugin で提供されている絵文字は、gitbucket-emoji-plugin で提供している画像に変換するように修正しました。

成果は、[v0.11.0](https://github.com/yasumichi/gitbucket-markdown-enhanced/releases/tag/v0.11.0) としてリリースしました。

### src/main/scala/io/github/yasumichi/gme/MarkdownEnhancedNodeRenderer.scala の修正

絵文字を処理する `renderEmoji` メソッドを定義しました。

```scala
  private def renderEmoji(
      node: Emoji,
      context: NodeRendererContext,
      html: HtmlWriter
  ): Unit = {
    val emojiBase = MarkdownEnhancedRenderer.EMOJI_BASE.get(context.getOptions())
    val providers = PluginRegistry().getSuggestionProviders.filter(_.id == "emoji")

    if (providers.length == 1) {
      val emojiList = providers(0).options(null)
      val emoji = emojiList.find(_._1 == node.getText().toString())
      if (emoji != None) {
        html
          .withAttr()
          .attr("src", s"${emojiBase}${node.getText()}.png")
          .attr("alt", s":${node.getText()}:")
          .attr("class", "emoji")
          .tag("img")
        html.tag("/img")
      } else {
        context.delegateRender()
      }
    } else {
      context.delegateRender()
    }
  }
```

分解して説明していきます。

```scala
    val emojiBase = MarkdownEnhancedRenderer.EMOJI_BASE.get(context.getOptions())
```

前項で導入した `EMOJI_BASE` の値を取得しています。

```scala
    val providers = PluginRegistry().getSuggestionProviders.filter(_.id == "emoji")
```

gitbucket-emoji-plugin が提供する SuggestionProvider を取得しています。

```scala
    if (providers.length == 1) {
      (中略)
    } else {
      context.delegateRender()
    }
```

取得した SuggestionProvider の数が 1 であれば、以下の処理をします。そうでなければ、EmojiExtension に処理を委譲します。

```scala
      val emojiList = providers(0).options(null)
```

取得した SuggestionProvider から、options メソッドを使用して絵文字のリストを取得します。

```scala
      val emoji = emojiList.find(_._1 == node.getText().toString())
```

絵文字のリストに node のテキストが含まれるか検索しています。

```scala
      if (emoji != None) {
        (中略)
      } else {
        context.delegateRender()
      }
```

絵文字が見つかれば、次の処理を行います。見つからなければ、EmojiExtension に処理を委譲します。

```scala
        html
          .withAttr()
          .attr("src", s"${emojiBase}${node.getText()}.png")
          .attr("alt", s":${node.getText()}:")
          .attr("class", "emoji")
          .tag("img")
        html.tag("/img")
```

gitbucket-emoji-plugin が提供する画像用の `<img>` タグを生成します。
