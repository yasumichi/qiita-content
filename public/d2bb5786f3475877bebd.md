## 概要

[GitBucket](https://gitbucket.github.io/) は、[GitHub](https://github.com/) のようにネットワーク上で [Git](https://git-scm.com/) リポジトリを共有できる Web サービスです。

- Apache-2.0 ライセンスで配布されている。
- [Scala](https://scala-lang.org/) で開発されており、Java 17 実行環境があれば、起動可能。
- お試しであれば、内臓の[H2データベースエンジン](https://www.h2database.com/html/main.html)が利用できる。
  - 実運用には、[PostgreSQL](https://www.h2database.com/html/main.html) や　[MariaDB](https://mariadb.org/) などのリレーショナルデータベースサーバーの利用を推奨
- 非常に簡単にセルフホストが可能です。

前回、Wiki にも [Ace](https://ace.c9.io/)(Ajax.org Cloud9 Editor) を導入しました。

今回は、これにマークダウン用のツールバーを導入しようという企みです。

## 現在考えているツールバーの仕様

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/7c6a8bcb-610e-4b85-8f22-8f4d2d38fe75.png)

大きくインライン記法用のボタンとブロック記法用のボタンに分類しています。

### インライン記法用

| ボタン | 説明 | 結果の例 |
| ------ | ---- | ------ |
| ![Bold](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/93eac73d-8f47-420a-a06a-abd5df58f311.png) | 選択文字を `**` で囲みます。 | `**選択文字**` |
| ![Italic.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/fa7226d9-a2b1-41a4-8fa7-975ae9625247.png) | 選択文字を `*` で囲みます。 | `*選択文字*` |
| ![code](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/94f1f53d-a172-4f4c-bd5c-ecb0b05bd391.png) | 選択文字をバッククオートで囲みます。 | &#96;選択文字&#96; |
| ![Link](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/6cc1e1b0-18f5-4ad6-bb69-b9ab45eaabcc.png) | 選択文字をリンクアンカーにします。 | `[選択文字]()`

現状、既に書式設定済みであるかまでは、関知していません。

### ブロック記法用

| ボタン | 説明 | 結果の例 |
| ------ | ---- | ------ |

| ![Quote](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/36738/ea21aeeb-cd3e-4c19-a4cc-9da9118e4d0e.png) | 選択範囲の行頭に `> ` を追加します。一段ネストができ、既にネストされている場合は、行頭文字をクリアします。 | `> 選択行` |

現状、行頭文字を追加するボタンは、他の行頭文字があると押したボタンの行頭文字に置き換える動きをします。
